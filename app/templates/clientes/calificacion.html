Como funciona la calificacion

Guia rapida para crear reglas sin enredarse.

Prioridad

Decide que regla se procesa primero cuando un cliente cumple varias al mismo tiempo.

Numero mas bajo = mas importante. `10` se aplica antes que `100`.

Ejemplo: una regla VIP con prioridad 10 y una regla de deuda con prioridad 100.

Reaplicar cada

Evita que la misma regla se aplique demasiado seguido.

Si pones `30`, esa regla solo podra volver a afectar al cliente despues de 30 dias.

Recomendado para reglas de sumar o restar estrellas.

Que significa cada campo

Nombre: es el titulo interno de la regla. Ejemplo: `Cliente VIP mensual`.

Metrica: que dato va a mirar el sistema. Ejemplo: cantidad de compras, monto gastado, dias desde la ultima compra.

Condicion: la comparacion. Ejemplo: `compras >= 5` o `dias desde ultima compra > 90`.

Periodo: se usa para metricas como compras o reparaciones. Ejemplo: `ultimos 30 dias`.

Accion: que debe hacer el sistema si la condicion se cumple.

Estrellas: cuantas estrellas asigna, suma, resta o limita esa accion.

Crear activa: si queda encendida apenas guardas la regla.

Acciones recomendadas

Asignar estrellas: deja al cliente exactamente en ese nivel. Ideal para categorias claras.

Sumar estrellas: aumenta desde el nivel actual. Conviene usar `Reaplicar cada` para que no suba demasiado rapido.

Restar estrellas: baja desde el nivel actual. Conviene usar `Reaplicar cada 15 o 30 dias`.

Limitar maximo: aunque cumpla otras reglas, no puede pasar de cierto nivel. Sirve mucho para clientes con deuda.

Exigir minimo: asegura un piso de estrellas para clientes valiosos o preferenciales.

Ejemplos utiles

Cliente VIP: Compras en ultimos 30 dias `>= 5` -> Asignar `5` estrellas.

Cliente tecnico frecuente: Reparaciones en ultimos 60 dias `>= 3` -> Asignar `4` estrellas.

Cliente inactivo: Dias desde ultima compra `> 90` -> Restar `1` estrella -> Reaplicar cada `30` dias.

Cliente con deuda: Saldo pendiente `> 0` -> Limitar maximo a `3` estrellas.

Consejo practico

Para empezar, te conviene crear pocas reglas: una para clientes muy activos, una para clientes inactivos y una para clientes con deuda. Asi se vuelve facil de entender y ajustar.
